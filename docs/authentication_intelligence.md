# EDITION — Authentication & Anti-Forgery Intelligence

> 真贋能力がなければブランドは成り立たない。AIによる贋作についてはあらかじめ徹底的に学習し、
> あらゆる技術とノウハウを得て、最新の技術動向も常にキャッチアップする。

---

## 1. 脅威の全体像：AI × 贋作の「軍拡競争」

### 1.1 AI が贋作制作を加速させる4つのベクトル

| ベクトル | 手法 | 脅威レベル | 対象カテゴリ |
|---------|------|-----------|------------|
| **スタイル模倣** | GANs/Diffusion で作家の「画風」を学習し、デジタル画像を生成。物理的な複製のガイドとしても使用 | ★★★★★ | 浮世絵、日本画、現代アート |
| **来歴偽造** | LLM で精巧な偽の売買契約書、鑑定書、所有履歴を生成 | ★★★★★ | **全カテゴリ** |
| **素材分析回避** | 歴史的に正しい顔料・素材を選定し、科学分析をパスする「物質レベルの贋作」をAIが最適化 | ★★★★☆ | 陶磁器、漆器、絵画 |
| **物理複製の精密化** | AIがCTスキャン + 3Dデータから構造を学習 → ロボティクスで精密再現 | ★★★☆☆（将来★★★★★） | 刀剣、彫刻、根付 |

### 1.2 日本刀特有の脅威

金沢工業大学（畝田道雄教授）の研究が示すもの：

- **機械学習で「評価される刀の特徴」を数値化**。展覧会図録10年分のビッグデータから、寸法・反り・姿のパターンを抽出
- AIが3次元設計図面を作成 → 刀匠と共に製作 → **展覧会入賞**を達成
- **TACHIプロジェクト**: ジェネレーティブデザインで「最適な刀剣形状」をAIが生成

**この技術が意味すること**：
> 善意の研究（伝統技術の継承・科学的理解）がそのまま「完璧な贋作の設計図」になり得る。
> CTスキャン + 成分分析のデータが公開されるほど、贋作者にとっての教科書が増える。

## 2. 世界基準の認証体制：サザビーズ / クリスティーズの方法論

### 2.1 三本柱（The Three Pillars）

#### Pillar 1: Connoisseurship（鑑識眼）
- 専門家パネルによる目視・触覚での評価
- 作家の「無意識の癖」（brushstroke fingerprint）を識別
- **Catalogue Raisonné**（全作品目録）との照合
- **日本刀における対応**: NBTHK（日本美術刀剣保存協会）の審査制度
  - 保存 → 特別保存 → 重要 → 特別重要の4段階認定
  - 銘の真偽（偽銘 gimei）判定、中心（なかご）の経年変化評価

#### Pillar 2: Provenance Research（来歴調査）
- 作品の所有履歴を遡及的に追跡
- オークション記録、展覧会図録、書簡、写真記録との照合
- **ギャップの特定**: 来歴が途切れている期間は贋作挿入のリスクが高い
- **日本文化資産**: 「伝来」の記録が命。大名家の蔵帳、茶会記録等

#### Pillar 3: Scientific Analysis（科学分析）
サザビーズが2016年に**Orion Analytical**（James Martin創設）を買収して以降、業界標準が引き上げられた。

| 技術 | 原理 | 検出対象 |
|------|------|---------|
| **XRF (蛍光X線分析)** | X線照射→元素固有の蛍光を検出 | 顔料・金属の元素組成。時代に合わない合成顔料を特定 |
| **MA-XRF (マッピング蛍光X線)** | XRFの面走査版 | 下描き・修復跡・後年の改変を可視化 |
| **FTIR (フーリエ変換赤外分光)** | 赤外光の吸収パターン | 有機分子（樹脂、バインダー）の特定 |
| **ラマン分光法** | レーザー散乱光 | 顔料・鉱物の分子レベルの同定 |
| **赤外反射撮影** | 近赤外光で表面下を透視 | 下描き、pentimenti（作家の修正）の検出 |
| **X線撮影/CT** | X線透過 | 内部構造、隠された改変、構造的異常 |
| **実体顕微鏡** | 高倍率表面観察 | 近代工具の痕跡、署名の重ね書き、構造異常 |

### 2.2 AI認証の最前線プレイヤー

| 企業/機関 | 技術 | 精度 |
|----------|------|------|
| **Art Recognition** (チューリッヒ) | CNN + Vision Transformer で画像から筆致分析 | コレクター・ギャラリー向け |
| **ArtDiscovery / Hephaestus** | 独自AI「Pictology」— 筆致の長さ・幅・曲率・空間関係を分析 | 保険裏付きの認証サービス |
| **Case Western Reserve大学** | 3Dトポグラフィースキャンで絵具の凹凸を解析 → 「無意識のスタイル」検出 | 研究段階 |
| **Acrylic Robotics** | AI + ロボットで公認レプリカを制作 → その過程で偽造パターンも学習 | 商業化中 |

## 3. EDITION が構築すべき認証フレームワーク

### 3.1 4層認証アーキテクチャ

```
Layer 4: Intelligence (市場異常検知)
  ├─ 出品パターンの統計分析（同一作品の複数出品検出）
  ├─ 価格異常の検出（不自然な低価格/高価格）
  └─ 来歴ネットワークの整合性チェック

Layer 3: AI Forensics (パターン認識)
  ├─ 筆致/工具痕の機械学習分析
  ├─ 刃文パターンのCNN分析（刀剣）
  ├─ 釉薬・土味のスペクトル分析データとの照合（陶磁器）
  └─ 木目・彫刻痕の3Dスキャン分析（彫刻/根付）

Layer 2: Scientific Verification (科学的検証)
  ├─ 素材の元素/分子分析（XRF, FTIR, ラマン）
  ├─ 放射性炭素年代測定（必要な場合）
  └─ 微細構造分析（SEM/EDS）

Layer 1: Connoisseurship + Provenance (人的検証)
  ├─ 専門家ネットワークによる目利き
  ├─ 来歴の文献調査
  └─ 比較対象としての参照データベース
```

### 3.2 カテゴリ別の認証優先技術

| カテゴリ | 最重要チェック | AI活用ポイント |
|---------|-------------|-------------|
| **刀剣** | 中心の経年変化、鍛え肌の一貫性、銘の書体 | 刃文パターンのCNN分類。鍛え肌の結晶構造パターン認識 |
| **陶磁器** | 土の成分、釉薬の組成、焼成温度の痕跡 | 窯跡データとの統計的照合。貫入パターンの経年劣化分析 |
| **浮世絵** | 摺りの鮮度、和紙の繊維、版木の磨耗度合い | 色彩分析と摺り番（初摺/後摺）の判定 |
| **日本画** | 絹本/紙本の素材、墨と顔料の組成 | 筆致の作家別フィンガープリント分析 |
| **漆器** | 漆の層構造（CT断面）、蒔絵の金粉粒度 | 装飾パターンの時代特定 |
| **根付** | 象牙/木の経年変化（patina）、彫刻工具痕 | 3Dスキャンからの工具痕パターン分析 |
| **着物** | 染料の化学組成、織りの密度 | 紋様データベースとの照合 |
| **金工** | 合金組成（赤銅、四分一の配合比）、彫金技法 | 象嵌パターンの流派判定 |

### 3.3 ブロックチェーン来歴管理

**Artory モデル（Christie's が採用）**：
- オブジェクト単位で改竄不可能なデジタルレジストリ
- 取引履歴、鑑定記録、科学分析結果を暗号学的に保全
- 所有者の匿名性を維持しつつ、来歴の透明性を確保

**EDITION への適用**：
- 全キュレーション資産にデジタル証明書を発行
- 科学分析結果のハッシュをチェーンに記録
- 「EDITION認証」というブランド価値の創出

## 4. 「AI vs AI」軍拡競争への戦略的対応

### 4.1 攻撃者の能力を常に上回るための原則

1. **攻撃者と同じ技術を理解する**（Offense-Informed Defense）
   - GANs/Diffusionモデルの最新アーキテクチャを把握
   - 「どう贋作を作るか」を知ることが最良の防御
   - **ただし**: 贋作制作そのものには関与しない（法的・倫理的リスク）

2. **検出不能な偽造は存在しない（現時点では）**
   - あらゆる生成AIには「ハルシネーション」的なアーティファクトがある
   - 周波数分析、Error Level Analysis（ELA）で検出可能
   - 物理的な贋作も、ナノレベルでは必ず「時間」の痕跡が異なる

3. **Multi-Modal Detection（複数手法の組み合わせ）**
   - 単一の検証手法は必ず突破される
   - 画像分析 + 素材分析 + 来歴調査 + 市場データの「四重チェック」
   - **Human-in-the-Loop**: 最終判断は必ず人間が行う

### 4.2 継続的学習の仕組み

- **学術論文の定期モニタリング**: arXiv (cs.CV, cs.AI), MDPI Sensors
- **業界動向**: Art Newspaper, Antiques Trade Gazette
- **オークションハウスの発表**: Sotheby's / Christie's の技術公開
- **日本国内**: NBTHK、日本刀剣博物館、国立文化財機構
- **新規スタートアップ**: Art Recognition, ArtDiscovery, Haltadefinizione

### 4.3 EDITIONの差別化ポイント

サザビーズやクリスティーズが**西洋美術**に最適化されているのに対し、
EDITIONは**日本文化資産**に特化することで以下の優位性を獲得できる：

- **専門データセット**: 日本刀の刃文データ、陶磁器の窯跡データなど、西洋のオークションハウスが持たないデータ
- **NBTHK/文化庁との連携可能性**: 国内機関との協力関係
- **来歴の日本語読解**: 蔵帳、箱書き、花押の解読はAI+日本語能力の組み合わせが必要
- **匠の暗黙知のデジタル化**: 刀匠、陶芸家の技法をAIで構造化

## 5. 高リスクシナリオと対策

### シナリオ A: AI設計の「完璧な」新作刀剣に偽銘が切られる
- **対策**: 中心の経年変化は物理法則に従う → 100年分の酸化膜は偽装不可能
- **AI活用**: 酸化膜の深さ・パターンをデータベース化し、年代推定AIを構築

### シナリオ B: 来歴書類をLLMで完全捏造
- **対策**: 文書のメタデータ分析（インク、紙、印刷技術の時代整合性）
- **AI活用**: 文体分析で「時代らしさ」の統計的検証

### シナリオ C: 3Dスキャン + AIで根付の完全複製
- **対策**: 象牙/木材の経年変化（patina）は分子レベルで異なる
- **AI活用**: 表面テクスチャの微細パターン認識（正規品DB vs 新品の統計差）

### シナリオ D: AI生成の「未発見の浮世絵」が市場に出回る
- **対策**: 和紙の繊維パターン、版木の磨耗一貫性、摺りの圧力分布
- **AI活用**: 既知の版木セットとの整合性チェック。存在しない版元の検出
